Refined Relocation Wiki

Welcome to the wiki!

Download

If you wish to download the latest builds of Refined Relocation, look here. These builds may crash, as they are snapshots, and may have unfinished features that will corrupt your world. Please make backups.

For stable versions, please see the forum.

Installation

Install forge for the correct version of the game you are runnning. Then, download the version of Refined Relocation that you wish to install and drop it into the mods folder. Report any bugs on the GitHub Issues page or on the forum.
